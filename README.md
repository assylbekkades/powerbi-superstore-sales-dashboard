# Power BI Superstore Sales Dashboard

This project demonstrates end-to-end analysis and visualization of retail sales data using Power BI.

## KPIs
- Total Sales
- Total Profit
- Profit Ratio
- Profit per Order
- Sales per Customer
- Average Discount
- Quantity Sold

## Key Visuals
- Monthly Sales Trend by Segment
- Monthly Sales Trend by Product Category
- Sales & Profit by Region
- Profit Ratio by State (Map)
- Top Customers by Sales
- Delivery Status Distribution by State

## Tools & Skills
- Power BI Desktop
- DAX
- Power Query
- Data Modeling
- Business Analytics
